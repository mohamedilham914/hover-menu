# Animated Hover Navigation Menu

A sleek, modern navigation bar where each menu item **grows + gets a colorful glowing background** on hover — built with **pure HTML & CSS** (zero JavaScript).

Live Demo → https://mohamedilham914.github.io/hover-menu/


*(Hover any item to see the smooth scale + background animation!)*

## Features

- Beautiful scale + background reveal on hover
- Smooth easing animation (feels premium)
- Fully responsive (works perfectly on mobile & desktop)
- Pure CSS — no JavaScript, no frameworks
- Lightweight (< 3 KB total)
- Easy to customize colors, speed, and text
- Perfect for portfolios, landing pages, agency sites
